Financial Formula

Net Present Value (NPV) Model:

\[ NPV = \sum_{t=1}^{T} \frac{C_t}{(1+r)^t} - C_0 \]

Where \(C_t\) = Net Cash Inflow, \(r\) = Discount Rate

Python Analysis


import numpy as np

def calculate_margin(revenue, cost):
    """
    Calculate Operating Margin
    """
    if revenue == 0:
        return 0.0
    return (revenue - cost) / revenue

# Q4 Data
print(f"Margin: {calculate_margin(1000, 400):.2%}")
