Valuation Models

We use the Net Present Value (NPV) formula for projections:

$$ NPV = \sum_{t=1}^{T} \frac{C_t}{(1+r)^t} - C_0 $$

Where $C_t$ is net cash inflow, $r$ is the discount rate, and $C_0$ is initial investment.

Explain the NPV formula: Sum of cash flows discounted back to present value minus initial investment.

Growth Calculation

Automated via Python script:


def calculate_cagr(start_val, end_val, periods):
    """
    Calculate Compound Annual Growth Rate
    """
    if start_val == 0:
        return 0
    return (end_val / start_val) ** (1/periods) - 1

# Q4 Analysis
cagr = calculate_cagr(100, 150, 3)
print(f"Growth Rate: {cagr:.2%}")
