Example — Calculating CAGR (Python)

def cagr(begin_value, end_value, periods):
    return (end_value / begin_value) ** (1.0 / periods) - 1

# example
print(cagr(100, 150, 3))  # ~0.1447 -> 14.47%

Key Financial Formula

Net Present Value (NPV):

$$\mathrm{NPV} = \sum_{t=0}^{T} \frac{C_t}{(1+r)^t}$$

Where $C_t$ = cashflow at time $t$, $r$ = discount rate.
